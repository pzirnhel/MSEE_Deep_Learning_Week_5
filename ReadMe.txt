To make use of this notebook:
-Download the Week5Repo, which creates the Week5Repo folder
-Create a "/tmp" folder inside the Week5Repo folder
-Download the gan-getting-started.zip archive from:
htts://www.kaggle.com/competitions/gan-getting-started/overview
-Unzip the gan-getting-started.zip archive, which creates a folder of the same name
-Transfer the MonetsGeneratingGANv5.ipynb jupyter notebook in it
-Open your anaconda environment, navigate to the Week5Repo folder and run the notebook
